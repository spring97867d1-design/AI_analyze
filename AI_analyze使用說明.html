🚀 YouTube AI 分析器 - 新手使用指南

歡迎使用本程式！這是一個能幫你自動下載 YouTube 音檔，並透過 Google Gemini AI 進行重點摘要與台股分析的自動化工具。請跟著以下步驟完成首次設定。

🛠️ 首次設定步驟

Step 1. 略過安全警告並執行
由於這是個人開發的程式，首次點擊執行檔 (EXE) 時，Windows 可能會彈出「Windows 已保護您的電腦」藍色警告視窗。這很正常！
👉 請點擊 「其他資訊」，然後按下 「仍要執行」。
Step 2. 產生設定檔 (config.json)
第一次順利執行程式後，程式會閃退（或是畫面提示你找不到設定檔），這時請看一下程式所在的資料夾，系統已經自動幫你建立了一個名為 config.json 的檔案。
Step 3. 填寫 Gemini API Key
程式需要有 Google 的「AI 鑰匙」才能幫你分析。請對著 config.json 點擊右鍵 ➔ 選擇「用記事本開啟」。
你會看到裡面有一行："api_key": "請在此填入你的_GEMINI_API_KEY"。
⚠️ 注意：必須將引號內的中文替換成你專屬的 API Key 才能執行程式！
Step 4. 如何免費取得 API Key？
請按照以下步驟免費申請你的 AI 鑰匙：
  1. 點擊下方按鈕前往 Google AI Studio 網站並登入你的 Google 帳號。
  2. 在左側選單點擊 「Get API key」。
  3. 點擊藍色按鈕 「Create API key」。
  4. 複製產生出來的那一長串英數字碼，貼回剛剛的 config.json 檔案中並存檔！
🔑 前往 Google AI Studio 申請 API Key

✨ 程式執行後會發生什麼事？

當你設定好 API Key 並再次點擊執行檔後，程式就會正式啟動！它的完整運作流程如下：

  • 開啟視窗： 會跳出一個介面，讓你貼上 YouTube 網址 (一次最多可貼 10 個)。
  • 下載音檔： 點擊開始後，程式會自動在背景將這些 YouTube 影片轉成高品質音檔下載。
  • AI 分析： 程式會將音檔上傳至 Google 雲端，並請 Gemini AI 依照設定好的規則 (整理重點、時間碼、台股操作建議) 進行分析。
  • 輸出報告： 分析完成後，會在資料夾中自動產生一份 .txt 純文字報告。
  • 自動歸檔： 最後，程式會很貼心地建立一個名為 分析完 的資料夾，把處理完的音檔移進去，保持你的桌面/資料夾乾淨整潔！

🎉 恭喜你！現在可以開始享受 AI 帶來的便利了！
